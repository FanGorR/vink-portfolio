图片放置说明 — VinK Portfolio
================================

把你的照片放在这个文件夹里，文件名要和 HTML 里写的一致。

建议格式：JPG，16:9 比例，宽度 1920px，压缩到 300KB 以下
压缩工具推荐：https://squoosh.app
