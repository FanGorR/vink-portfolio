Assignment 1 — Global Film Industry Dashboard

Analysed and built an interactive Power BI dashboard of the global film industry (1980–2015) using the Kaggle Movies Dataset (30,000+ films).

Key insight from the data: global box office grew from $2B (1980) to $28B (2015); US contributed 67%+ of total revenue.

Assignment 3 — Mobile AR Data Visualisation

Built a mobile AR prototype using Unity + AR Foundation + ARCore that projects smart factory IoT sensor data as 3D visualisations into real physical space. Stable 60 FPS on mid-range devices.

AR Visualisation — Technical Details

Data Pipeline

  • CSV parser (CSVReader) → MachineData OOP model → LINQ query aggregation → visualisation generation
  • 35 machines · 1,050 sensor records · 13 variable dimensions
  • Colour coding follows ISO 3864 standard (green/yellow/red)

Visualisations & Interaction

  • Temperature bar chart (height + colour encoding)
  • 5×5 downtime risk heatmap (colour intensity)
  • Vibration level bar chart
  • Multi-touch: tap-to-place, pinch-to-scale, two-finger rotation
  • 3 analytical panels with 3–5 second auto-refresh
