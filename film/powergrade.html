The Problem It Solves

"I used to be a DaVinci Resolve free user hunting for a genuine film look online. Every tutorial I found required paid plugins or Studio-exclusive features — the effects I actually wanted were locked behind paywalls or dependencies I couldn't afford."

So I built one. Six months of research into DaVinci Resolve's native node system — halation, bloom, film grain, glow, colour response curves — all rebuilt from scratch using only tools available in the free version of DaVinci Resolve.

The result: a complete cinematic film look system that supports 10 camera log formats across a clean 18-node pipeline, with a full PDF guidebook included.
